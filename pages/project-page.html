RaspPi Remote Security Camera

A Raspberry Pi based security camera that can be controlled remotely via a web app.


The camera is controlled via a web app that is hosted on the Raspberry Pi. The web app is built with Flask and uses the picamera library to control the camera. The web app is also used to view the live stream from the camera.


OpenCV is used to detect motion in the live stream and send an email notification, and a text notification, when motion is detected. The notification includes a snapshot of the object that caused the trigger.
